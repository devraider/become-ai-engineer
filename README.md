# 🤖 Become an AI Engineer

> **A 14-week bootcamp to transform into a job-ready AI Engineer**

This training program covers everything you need to pass AI engineering interviews and build production AI systems - from data foundations to agentic AI.

## 🚀 Quick Start

1. Read the [**Roadmap**](ROADMAP.md) to see the full curriculum
2. Start with [**Week 1**](week-1/README.md) - Environment Setup
3. Complete each week's project before moving on

## 📅 Curriculum

| Phase             | Weeks | Focus                                     |
| ----------------- | ----- | ----------------------------------------- |
| Foundation        | 1-3   | Setup, Data, ML Fundamentals              |
| Deep Learning/LLM | 4-6   | PyTorch, LLM APIs, HuggingFace            |
| Advanced AI       | 7-9   | Transformers, RAG, Vector DBs             |
| Agentic AI        | 10-14 | LangGraph, HF Advanced, MCP, Vectors, ADK |
