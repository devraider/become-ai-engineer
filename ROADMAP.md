## 🏗️ Learning Structure (Each Week)

Every week follows the same structure for efficient learning:

```
📖 Concepts (30 min read)
    ↓
💻 Code Examples (1-2 hours hands-on)
    ↓
✅ Mini Exercises (30 min practice)
    ↓
🚀 Weekly Project (2-4 hours build)
    ↓
📝 Interview Questions (review & memorize)
    ↓
✓ Takeaway Checklist (self-assessment)
```

## 🎓 How to Use This Bootcamp

### For Python Developers

- Skip Python side-learning files
- Focus on AI-specific content
- You'll move faster through Weeks 1-2

### For Complete Beginners

- Start with `side-learning/python-basics.md` first
- Don't rush - understanding is more important than speed
- Run every code example yourself

### For Team Training

- Weekly 2-hour sessions recommended
- Pair programming on projects
- Present projects to each other for practice
